This is a legacy case study of ALiBi when Flashlight/TorchInductor generated kernel was not as performant.

The purpose of keeping this legacy is to help understand the difference of Flashlight/TorchInductor generated kernel vs FlexAttention kernel.

`kernel_alibi.py` is the (old) kernel generated by Flashlight/TorchInductor with detailed per-line comment that may refer line numbers in `kernel_flex_alibi.py`, the FlexAttention kernel.

The config here is batch_size=32, nheads=32, seqlen=512, headdim=64
